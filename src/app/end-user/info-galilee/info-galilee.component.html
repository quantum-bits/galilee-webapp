What is the Galilee Project?

The Galilee Project is an interactive Scripture Engagement website that can be used by individuals or groups to help people engage with Scripture so that they can engage with God.

Using the Revised Common Lectionary reading plan, each Bible reading is provided with multiple Scripture engagement practices to choose from to help the reader meditate/reflect on Scripture. The Scripture engagement practices can be done either individually or in a group. Participants can keep a private journal on the website and/or can post comments about the readings and practices that can be seen only by self-selected group members.

The Galilee Project was developed by the Taylor University Center for Scripture Engagement partnering with Bible Gateway.
