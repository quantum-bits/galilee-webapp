How do I use the Journal?

Welcome to the Journal feature of Galilee. Journaling your thoughts, prayers, commitments, fears, dreamsâ€¦whatever comes to you as you read scripture, is a powerful tool for spiritual growth. The Journal feature of Galilee is a completely private tool; no one can see what you write here besides you. Using the calendar feature you can go back to read previous journal entries, a very helpful tool to help you see what God has been doing in your life. You can even "tag" specific entries with any label you'd like (e.g., prayer requests, commitments, people, topics) so that you can more easily find journal entries that you'd like to return to.
